Chat Widget Integration Demo

This is a demo page showing how to integrate the chat widget into any website.

Look at the bottom-right corner - you'll see the chat widget floating button!

Click on it to open the chat window and test the functionality.

How to integrate:

  1. Copy the embed.js file to your website
  2. Add this script tag to your HTML:
    3. <script src="path/to/embed.js"></script>
  3. Update the configuration in embed.js with your Firebase and webhook URLs
  4. That's it! The chat widget will appear automatically
